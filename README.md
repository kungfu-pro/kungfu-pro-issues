# Kung Fu Pro 

下载:  https://github.com/kungfu-pro/kungfu-pro-issues/releases

开箱即用的代理解决方案 <br>
灵活的规则配置，兼容多种代理协议 <br>
DNS 劫持方案支持大型网络环境，不影响正常流量 <br>

[使用](https://kungfu-pro.github.io/doc/#/?id=%E6%A1%88%E4%BE%8B%E4%BD%BF%E7%94%A8)
